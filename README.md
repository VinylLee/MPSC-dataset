# MPSC-dataset
This repository contains a collection of smart contracts under test used in our research on smart contract analysis and testing.

The dataset is constructed based on contracts collected from the SmartBugs Curated dataset, a widely used benchmark for evaluating smart contract security analysis tools.

# Dataset Source

The smart contracts in this repository are derived from:

SmartBugs Curated
https://github.com/smartbugs/smartbugs-curated

SmartBugs Curated is a curated dataset of vulnerable smart contracts collected from real-world projects and academic benchmarks. It has been widely adopted in smart contract security research.

In this repository, we select and organize contracts from this dataset to support our experiments.
